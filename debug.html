🛠️ Solusi Cepat

❌ Jika CMS Backend tidak jalan:
1. Buka Command Prompt / Terminal
2. Jalankan: cd d:\alphonsus-portfolio-website-design\cms
3. Jalankan: npm run dev
4. Tunggu sampai muncul "CMS Backend running on port 5001"
🗄️ Jika MongoDB tidak running:
1. Tekan Windows+R, ketik: services.msc
2. Cari service "MongoDB" dan klik Start
Atau jalankan di Command Prompt: net start MongoDB
📭 Jika Database kosong:
1. Buka terminal di folder CMS
2. Jalankan: npm run seed
3. Ini akan populate database dengan sample data
🚀 Quick Start (Termudah):
Dobel-klik file: QUICK_START.bat di folder project Anda
